Pre-launch checklist
Backups verified + one successful restore drill · alerts firing correctly · TLS enforced end-to-end · indexes created from query patterns · smoke + load test passed · secrets in vault, not env files.
Two decisions that save real money & effort
Create indexes up front from your query patterns (cases by ward / status / date, lookups by citizen) — cheap now, painful to retrofit under load. Keep photos in object storage, never in Mongo — keeps the DB small so you stay on small, cheap nodes.
