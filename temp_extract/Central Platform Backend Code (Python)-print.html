Central Platform — FastAPI reference code
Illustrative Python for the multi-tenant Central Platform, matching your main.py + app/ layout. FastAPI · SQLAlchemy 2.0 · PostgreSQL row-level security · OTP + JWT · FCM/APNs. Reference patterns to copy into the repo, not a drop-in build.
How to read this
Each block is a file in your app/ tree. Every module follows the same shape — router.py (routes), service.py (logic), models.py (tables), schemas.py (Pydantic). Tenant isolation lives in common/ and applies to every request.
Two structure decisions, resolved: Customer Management (the paying orgs) lives inside subscriptions/ as the Account model, since billing owns the commercial relationship. directory/ is the registry (which tenants exist + config, read-heavy, cached); tenants/ holds per-tenant runtime data (categories, roles, wards).
